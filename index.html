Portfolio

TiledLoader

Unity editor extension for loading Tiled Map Editor files into Unity, providing a 2D, tile-based workflow for designing both 2D and 3D levels.

Terrain Generator

Unity editor extension for generating procedural terrains both in the editor and at runtime. Capable of generating heightmaps and applying textures, details, and trees using a configurable, rule-based system.

Food Feud

Tower Defense game in which the player must defend their burger joint from being overrun by Tacos.

Lit Limo

Classic GTA-like game in which the player runs down pedestrians.

Dear Leader

In-development city-building game made in Unity.

Smash Game

2D Multiplayer fighting game similar to Super Smash Bros.

PyZelda

Implementation of The Legend of Zelda: Link's Awakening in a custom game engine built on top of PyGame in Python.

Pathfinders

2-player VR dungeon crawler in which the VR player explores a dungeon while the second player uses the keyboard, mouse, and computer monitor to control an overhead camera to help guide the VR player, and a terminal to manipulate the dungeon.

Kamikazee

Experimental 3D game made during a 12 hour game jam in which progression is linked to taking advantage of physics bugs common in other games.
